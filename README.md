### 🚀 IT Skill Demand Analysis Framework

An AI-powered platform that analyzes IT job postings to identify trending skills, salary insights, and workforce demand using Natural Language Processing (NLP).

### ✨ Key Features

* 🔍 Automatic Skill Extraction from job descriptions

* 📈 Trending Technology Analysis (AI, Cloud, DevOps, etc.)

* 💰 Salary & Experience Insights

* 🏷️ Domain Classification of technical skills

* 📊 Interactive Dashboards & Visualizations

* 📄 PDF and CSV Report Generation

### ⚙️ How It Works

* Load Dataset

  * Import IT job postings with titles, descriptions, skills, salary, and experience details.
* NLP Processing

  * Clean and preprocess text using tokenization and stopword removal.
* Skill Extraction

  * Identify technical skills automatically from job descriptions.
* Domain Classification

  * Group skills into domains like AI/ML, Cloud Computing, DevOps, Frontend, Backend, and Cybersecurity.
* Analytics & Visualization

  * Generate trend analysis, salary comparisons, heatmaps, and interactive charts.
* Report Generation

  * Export insights in PDF and CSV formats for easy sharing and analysis.

### 🏗️ System Flow

IT Job Postings → NLP Processing → Skill Extraction → Domain Classification → Analytics Dashboard → Reports

### 🛠️ Tech Stack

* Frontend: Streamlit

* Backend: Python

* NLP & Data Processing: Pandas, NLTK, Scikit-learn, TF-IDF

* Visualization: Plotly, Matplotlib

* Reporting: PDF & CSV Export

### 🎯 Benefits

* Students – Learn in-demand skills and plan career paths

* Recruiters – Understand hiring trends and salary benchmarks

* Educational Institutions – Align curriculum with industry needs

* Analysts & Policymakers – Gain workforce and technology insights

### 🌟 Outcome

This framework transforms unstructured IT job postings into actionable workforce intelligence, helping users make smarter decisions in an AI-driven job market.
